# Change Log

All notable changes to the "Hipster Green Theme" extension will be documented in this file.

## [1.1.0] - 2025-11-11

### Added
- **Enhanced semantic highlighting** for TypeScript, Python, Rust, and C/C++
- **Improved bracket pair colorization** with 6 distinct colors for nested brackets
- **Better Git integration** with enhanced diff visualization and merge conflict colors
- **Comprehensive debugging experience** with colored breakpoints, call stack, and console output
- **Enhanced minimap and overview ruler** indicators for better code navigation
- **Improved find/replace widget** styling and search result highlighting
- **Breadcrumb navigation** styling for better file path visibility
- **Symbol icon colors** for better IntelliSense and outline view
- **Enhanced notification** styling for better user feedback
- **Comprehensive editor enhancements** including sticky scroll, marker navigation, and resize borders

### Enhanced
- Terminal integration with better cursor and background styling
- Welcome page theming for consistent experience
- Diff editor with better line and text highlighting
- Merge conflict resolution with distinct header and content backgrounds
- Git decoration colors for better repository status visualization

### Technical Improvements
- Added semantic token color customizations for language-specific syntax
- Improved color contrast and accessibility
- Enhanced font styling for different code elements (bold, italic, strikethrough)
- Better support for modern VS Code features

## [1.0.0] - 2025-11-11

### Added
- Initial release of Hipster Green Theme for VS Code
- Complete port from Zed theme format
- Dark theme with retro terminal aesthetic
- Comprehensive syntax highlighting for all major programming languages
- Terminal colors that match the editor theme
- Optimized UI colors for VS Code interface

### Features
- Green-dominant color palette with high contrast
- Support for all VS Code language features
- Consistent theming across editor, sidebar, terminal, and other UI components
- Carefully selected colors for optimal readability and reduced eye strain

## Color Scheme Details

The theme is based on the classic Hipster Green terminal color scheme with modern adaptations for VS Code:

- **Editor Background**: Deep dark brown-black (#100a05)
- **Primary Text**: Bright green (#84c138)  
- **Syntax Highlighting**: Vibrant greens, cyans, and contrasting colors
- **UI Elements**: Consistent green accent colors throughout

This release provides a complete and polished VS Code theme experience.
